## [0.1.0] - 2025-11-08

### Added
- Initial repository setup
- PlatformIO build configuration for ESP32/ESP8266
- GitHub Actions CI/CD workflows
- Basic project structure
- Web UI skeleton (HTML/CSS/JS)
- README with comprehensive documentation
- MIT License
- Contributing guidelines

### Firmware Variants
- `esp32-nrf24` - ESP32 + NRF24L01+ (Hoymiles HM)
- `esp32-dual` - ESP32 + NRF24 + CMT2300A (HM + HMS/HMT)
- `esp32s3-dual` - ESP32-S3 + NRF24 + CMT2300A (latest hardware)
- `esp8266-nrf24` - ESP8266 + NRF24L01+ (budget option)
